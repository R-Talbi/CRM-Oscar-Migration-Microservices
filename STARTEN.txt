

PROJEKT STARTEN


VORAUSSETZUNGEN:
- Docker Desktop installiert & gestartet
- Git installiert



SCHRITT 1: Projekt holen

git clone [DEIN_REPO_URL]
cd microservices-migration


SCHRITT 2: Starten (sleep 60!!!!!!!!!)

docker-compose up -d
(Warten bis alle Container laufen)


SCHRITT 3: Testen direkt in VS oder in Git Beispiel für E2E-Tests

python tests/E2E-Tests.py


SCHRITT 4: BROWSER-Tests zur Prüfung der CRM-Oscar und auch neue Microservices

  http://localhost:8000/dashboard/
  http://localhost:8000/admin/
